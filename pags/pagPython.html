    O que é python?

    Python é uma linguagem de programação popular. Foi criado por Guido van Rossum e lançado em 1991.

    É usado para:

    • desenvolvimento web (lado do servidor),
    • desenvolvimento de software,
    • matemática,
    • script do sistema.
