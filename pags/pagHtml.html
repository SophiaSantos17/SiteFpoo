    O que é HTML?


    HTML é uma linguagem de marcação usada na construção de páginas web e seus documentos podem ser interpretados por navegadores web (Google Chrome, Microsoft Edge etc)




    TAGS no html


    TAGS são estruturas da linguagem de marcação q contém instruções e uma marca de início e fim, e podem apresentar um elemento dentro do texto como título, lista, tabela, parágrafo, entre outros. Elas são indicadas por ('<' '>').
    Sintaxe: < (tag) > < / (tag) >




    Elemento head


    O elemento head no HTML providencia informações gerais sobre o documento, que não serão exibidas, como tipo, linguagem, nome do documento, link para a linguagem que vai executar os comandos e funções, como JavaScript, Python, PHP.
